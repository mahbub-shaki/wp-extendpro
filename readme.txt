== Description ==
A professional reusable WordPress plugin that works with any theme.
WP ExtendPro provides advanced functionality that can be easily integrated 
into any WordPress site without coding. Features include Auto Mode and 
Dashboard Settings UI for quick configuration.

== Installation ==
1. Upload the `wp-extendpro` folder to the `/wp-content/plugins/` directory, 
   or install the plugin directly via the WordPress Plugins screen.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'Settings -> WP ExtendPro' to configure options.

== Frequently Asked Questions ==

= How do I enable Auto Mode? =
Auto Mode is enabled by default. You can check or toggle it in the 
WP ExtendPro settings page under Dashboard Settings.

= Can I use WP ExtendPro with any theme? =
Yes! The plugin is designed to work with any WordPress theme.

= Does WP ExtendPro require additional plugins? =
No, it works independently without requiring other plugins.
